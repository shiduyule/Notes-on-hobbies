## 快捷键:

**注释**:**ctrl + /**  

选中文本之后, 按下 **Ctrl + B** 可以给选中文本**加粗**.

同理 *Ctrl + I* 可以让选中文本变为*斜体*
(这和word中的B和I对应的功能是一致的)

选中文本后，按下快捷键 **Ctrl + [** 和 **Ctrl + ]** 向左或者向右缩进

**粘贴图片**：按下快捷键 **Ctrl + Alt + V**
