# 快捷键 ctrl + M （施加两次是公式块）

>Markdown 的数学公式吸纳了大部分的 Latex 语法, 你可以以一种简单的方式在 VS Code 中书写数学公式.


**行内公式:** 

单位圆 $x^2+y^2=1$

**公式块:**

$$
\begin{cases}
x=\rho\cos\theta \\
y=\rho\sin\theta \\
\end{cases}
$$


>VS Code 有着非常便捷好用的自动补全功能, 只需要简单地打出你想打的内容的几个字母 (乱序也行), 再使用 ↑ ↓ 进行选择, 最后按下回车就可以打出你想要的内容.
再使用 HyperSnips 插件, 就能够写得飞快
